# Role Name: RabbitMQ

An Ansible Role that installs RabbitMQ on RedHat/CentOS.

## Requirements

None.

## Role Variables

### `defaults/main.yml`

* `rabbitmq_cluster: False`
* `rabbitmq_erlang_cookie: OMZWYPRFPJDAUGKDSXAP`
* `rabbitmq_cluster_master: localhost`
* `rabbitmq_users:`
    ```
      - user: admin
        password: admin
        vhost: /
        configure_priv: .*
        read_priv: .*
        write_priv: .*
    ```

* `rabbitmq_users_removed:`
    ```
      - guest
    ```

## Dependencies

None.
